# QuartzHost
Quart.Net 在dotnetcore3.1平台的实现  
![GitHub](https://img.shields.io/github/license/cddldg/QuartzHost)

## 主要实现
任务管理：启动、停止、新建、暂停、恢复等;   
ASP.NET Core Blazor展现（进行中）;  
任务支持参数;    
全API支持;   
## 特点
任务插件式运行;  
实现Job不需要引用Quart.Net;  
任务支持异步;   
集成NLog;   
可多节点部署;
